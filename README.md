## Udemy Clone Website

This project is a simple Udemy-inspired online learning platform UI built using HTML and CSS. It replicates the basic layout and design of a course-based website, focusing on clean structure and responsive styling.

## Features
Homepage with course listings
Navigation bar with categories
Course cards with images, titles, and pricing
Responsive design for different screen sizes
Clean and user-friendly interface

## Technologies Used
HTML5
CSS3
